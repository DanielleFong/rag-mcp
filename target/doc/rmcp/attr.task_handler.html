Attribute Macro task_handler 

#[task_handler]
Expand description

ยงtask_handler

Generates basic task-handling methods (enqueue_task and list_tasks) for a server handler using a shared [OperationProcessor]. The default processor expression assumes a self.processor field holding an Arc<Mutex<OperationProcessor>>, but it can be customized via #[task_handler(processor = ...)]. Because the macro captures self inside spawned futures, the handler type must implement [Clone].
