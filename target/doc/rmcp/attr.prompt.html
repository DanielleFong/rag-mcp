Attribute Macro prompt 

#[prompt]
Expand description

§prompt

This macro is used to mark a function as a prompt handler.

This will generate a function that returns the attribute of this prompt, with type rmcp::model::Prompt.

§Usage

fieldtypeusage
nameStringThe name of the prompt. If not provided, it defaults to the function name.
descriptionStringA description of the prompt. The document of this function will be used if not provided.
argumentsExprAn expression that evaluates to Option<Vec<PromptArgument>> defining the prompt’s arguments. If not provided, it will automatically generate arguments from the Parameters<T> type found in the function signature.
