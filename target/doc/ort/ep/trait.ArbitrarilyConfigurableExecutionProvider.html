Expand description

Trait used for execution providers that can have arbitrary configuration keys applied.

Most execution providers have a small set of configuration options which don’t change between ONNX Runtime releases; others, like the CUDA execution provider, often have options added that go undocumented and thus unimplemented by ort. This allows you to configure these options regardless.
